# Ytdt
 A poorly written Youtube video downloader

## Usage
run `ytdt` after finishing the installation and it'll give you a few options for the download which include the following:
- The video's URL or a search query to search for the video within the tool
- Download path
- File format
(note: If you used a search query it gives you the option to select one of the first 5 results)
## Installation
### Clone the repo
 ```bash
 git clone https://github.com/posaidon09/ytdt
 ```
### Install the packages
 ```bash
 npm install
 ```
### Setup the CLI (you may have to use sudo if you're on linux)
 ```bash
 npm i -g
 ```
